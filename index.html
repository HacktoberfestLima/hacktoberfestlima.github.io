Sobre el Hacktoberfest

Open source está cambiando el mundo – un pull request por vez.

Hacktoberfest® es una celebración que dura todo el mes de octubre que festeja el software de código abierto, este evento es realizado por DigitalOcean y DEV.

Mas información hacktoberfest.digitalocean.com

Networking y registro de participantes

Networking con bocaditos mientras esperamos que comience el evento (habra SWAG para los primeros que lleguen)

09:30

Mi viaje en el Open Source Eddú Meléndez

Se hablara de como en el mundo de hoy el Open Source consume al mundo y como las empresas empiezan a elegir proyectos open source para sus desarrollos en lugar de lidiar con licencias. Se explicara como grandes empresas como IBM, Microsoft también forman parte del cambio al liberar diferentes proyectos hacia la comunidad.
